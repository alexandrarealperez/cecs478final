# Network Intrusion Detection System

# Overview
This project implements a Network Intrusion Detection System in C programming using libpcap. The system will capture and analyze network traffic to detect suspicious activity such as port scans and SYN flood attacks.

# Goals
- Capture network packets using libpcap
- Detect basic attack patterns (port scans, SYN floods)
- Log alerts for suspicious activity
